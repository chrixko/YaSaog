YaSaog
======

Bubble themed game for my admission assignment

Controls
=======
Mouse		-> Move hair dryer

Left Click 	-> Blow

P 			-> Pause / Unpause

R 			-> Restart level

Escape		-> Back to menu


F12			-> Debug overlay
